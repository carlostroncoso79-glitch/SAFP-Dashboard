# SAFP-Dashboard

## Sistema Alimentario Familiar Permanente

Dashboard interactivo para gestionar la alimentación familiar basado en el manual SAFP.

### 🚀 Características
- Gestión de perfiles familiares
- Generación automática de menús mensuales
- Lista de compras inteligente
- Exportación a PDF
- Autenticación segura con Supabase

### 🛠️ Tecnologías
- HTML5 + CSS3 + JavaScript Vanilla
- Supabase (Auth + PostgreSQL + RLS)
- GitHub Pages (Hosting)

### 📦 Instalación
1. Clonar repositorio
2. Configurar variables de entorno en Supabase
3. Ejecutar schema SQL
4. Desplegar en GitHub Pages

### 🔐 Configuración de Supabase
1. Crear proyecto en Supabase
2. Copiar URL y API Key
3. Configurar en `assets/js/supabase.js`
